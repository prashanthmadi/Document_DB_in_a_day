# Document DB in a Day (Draft)

A one-day, AI-focused workshop to help participants learn DocumentDB alongside Microsoft Foundry. This is a draft version and will be updated with additional modules.

## Workshop Goals

- Learn DocumentDB fundamentals and NoSQL concepts
- Build hands-on experience with vector search and AI scenarios
- Connect DocumentDB with Microsoft Foundry workflows

## Modules (Current Draft)

### 1) Introduction
- Overview of DocumentDB and the workshop structure
- Setup guidance and expectations

### 2) NoSQL Core Concepts
- NoSQL fundamentals
- Schema design patterns
- Data relationships

### 3) AI Vector Search
- DiskANN vector indexing and search
- Healthcare industry example with pre-generated embeddings

### 4) AI Agents with DocumentDB
- Planned module for creating agents with DocumentDB and Foundry
- In progress (draft)

## What’s Included

- Pre-generated embeddings and sample data
- Step-by-step README files in each module
- Optional scripts showing how embeddings were generated

## Status

This repository is a draft. More modules and refinements are coming, including expanded AI agent scenarios and Foundry integration examples.
